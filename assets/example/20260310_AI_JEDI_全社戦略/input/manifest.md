# Manifest

## 何を作る案件か

AI時代の事業計画を伝えるホームページ（ランディングページ）。
第一回顧客ミーティング用のオンライン共有ページとしてまず作成し、後にホームページとして正式公開する。

## 誰向けか

製造業向けDXコンサルティングサービスを提供している企業（見込み顧客）。
生成AI関連PJTを複数抱えており、開発委託先の品質に課題を感じている。

## いつまでに必要か

第一回顧客ミーティングまでに最初のページが必要。

## 何が決まっているか

- 第一回ミーティングはオンライン形式
- ページを画面共有しながらミーティングで使う
- 伝えたいサービス内容：人材紹介、自社開発（委託受託）、研修・人材育成
- 段階的に拡充：アセット（会社資料等）を追加しながらコンテンツを育てていく
- 最終的にホームページとして公開

## 何が未確定か

- アセット（会社資料・サービス詳細・実績など）はまだ未作成・未提供
- 正式公開のデプロイ先
- 自動デプロイの要否
- ページのデザイン詳細
